# Comfort Time — лендинг

Профессиональный одностраничный сайт агентства недвижимости в Астане.

## Стек

- **Tailwind CSS** (Play CDN) — утилитарные стили + кастомная палитра бренда
- **AOS** — плавные анимации появления при скролле
- **Lucide Icons** — современные SVG-иконки (без эмодзи)
- **Google Fonts** — Manrope (заголовки) + Inter (текст)
- Vanilla JS — маска телефона, счётчики, форма → WhatsApp, бургер-меню

Без сборки и npm. Открывается двойным кликом по `index.html`.

## Структура

```
comfort_time/
├── index.html      # Разметка
├── styles.css      # Кастомные стили (поверх Tailwind)
```

## Формы

Обе формы (в hero и в секции «Контакты») при отправке открывают WhatsApp с уже
подставленным сообщением — заявка приходит вам в чат. Чтобы подключить отправку
на email/Telegram-бот/CRM, замените `window.open(buildWaLink(...))` в
[script.js](script.js) на `fetch('/api/lead', { ... })`.

## Локальный запуск

Просто откройте `index.html` в браузере. Для корректной работы карты и шрифтов
нужен интернет.

Опционально, чтобы запустить через локальный сервер (PowerShell):

```powershell
# Python
python -m http.server 5500
# или Node
npx serve .
```
